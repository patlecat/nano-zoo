# User-Requirements

Stand: 2026-03-16

## Ziel
Dieses Dokument beschreibt, was Kinder fuer den Hackathon auf ihrem Laptop dabeihaben oder am Veranstaltungstag einrichten sollen.

## Zielgruppe
- Kinder mit eigenem Laptop, oder Laptop vom Hackathon + Maus
- Betriebssystem: Windows oder Debian Linux

## Muss vor Ort moeglich sein
- Laptop laeuft stabil und hat Netzteil dabei
- WLAN-Verbindung funktioniert
- Webbrowser ist nutzbar (z. B. fuer GitHub)
- E-Mail-Zugang funktioniert (wichtig fuer Konto-Erstellung/Verifizierung)

## Empfohlene Vorbereitung vor dem Hackathon

### 1) GitHub-Konto
- Optimal: bereits vorhandenes GitHub-Konto
- Alternativ: Konto am Tag selbst erstellen
- Wichtig:
  - Zugriff auf E-Mail fuer Verifizierung
  - Passwort bekannt und sicher gespeichert

### 2) Cursor-Editor
- Empfohlen: Cursor IDE ist bereits installiert
- Falls noch nicht vorhanden: Installation am Hackathon-Tag
- Nach Installation kurz pruefen:
  - Editor startet
  - Ein Ordner/Projekt kann geoeffnet werden

### 3) Grundlegende Laptop-Bereitschaft
- Genug freier Speicherplatz fuer Tools und Projektdateien
- Akku in Ordnung, Netzteil dabei
- Admin-Rechte oder Hilfe durch Eltern/Betreuung verfuegbar (falls Installation blockiert ist)

## Einrichtung am Veranstaltungstag

### Pflicht-Checkliste
- [ ] Laptop + Netzteil vorhanden
- [ ] WLAN verbunden
- [ ] Browser funktioniert
- [ ] E-Mail-Zugang funktioniert

### Entwicklungs-Checkliste
- [ ] GitHub-Konto vorhanden oder neu erstellt
- [ ] Cursor installiert
- [ ] Projekt kann im Editor geoeffnet werden

## Hinweise je Betriebssystem

### Windows
- Installation von Programmen kann Admin-Rechte erfordern
- Vorab sicherstellen, dass Installationen nicht durch Schul-/Familienrichtlinien blockiert sind

### Debian Linux
- Installation je nach Distribution/Paketquelle vorbereiten
- Sicherstellen, dass Benutzerkonto Software installieren darf

## Empfehlung fuer Betreuende/Eltern
- Beim ersten Login (GitHub, E-Mail) in Reichweite bleiben
- Bei Passwort-Reset oder Verifizierung helfen
- Keine sensiblen Zugangsdaten mit anderen teilen

## Ergebnis nach erfolgreicher Einrichtung
Wenn alles klappt, hat jedes Kind:
- ein nutzbares GitHub-Konto (bestehend oder neu),
- einen installierten und startbereiten Cursor-Editor,
- einen lauffaehigen Laptop fuer die Hackathon-Aufgaben.
